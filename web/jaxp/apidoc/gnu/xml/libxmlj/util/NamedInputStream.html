Class NamedInputStream

public class NamedInputStream
extends FilterInputStream
An input stream associated with an XML system ID. It can report the system ID and the first few bytes of the stream in order to detect the character encoding of the stream.
Author:
Chris Burdess

Method Details

getDetectBuffer

public byte getDetectBuffer()

Returns the first few bytes of the stream for character encoding purposes. The entire stream can thereafter be read normally from the beginning. This method is only valid if no bytes have yet been read from the stream.

Throws:

IOException
