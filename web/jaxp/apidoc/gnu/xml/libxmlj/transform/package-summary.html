Package gnu.xml.libxmlj.transform

A JAXP-compliant wrapper for the XSLT C library for Gnome, also known as libxslt.

Class Summary

GnomeTransformer An implementation of Transformer which performs XSLT transformation using libxslt.
GnomeTransformerFactory An implementation of TransformerFactory producing Transformer objects which use libxslt for transformation.
TransformTest Simple test class with command line interface.
A JAXP-compliant wrapper for the XSLT C library for Gnome, also known as libxslt. Allows to use libxslt via the Java API for XML processing.

Usage:

  • Add libxmlj.jar to your CLASSPATH.
  • Add libxmlj.so to your LD_LIBRARY_PATH.
  • Set the system property javax.xml.transform.TransformerFactory to gnu.xml.libxmlj.GnomeTransformerFactory.
