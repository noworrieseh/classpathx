provides a free implementation of the java beansTM activation framework.

The JAF links beans and internet standards together.

Important files

Standard mime.types files are parsed by the JAF MIME handling classes to work out the mime types available on the local system.

Standard mailcap files (defined in RFC ) are parsed by mailcap handling classes to work out what beans should be linked to what mime types.
